Math 333 Learning Outcomes and Practice Problems


First Order Differential Equations
  • LO1: I can show a given function is in the family of solutions to a differential equation, with or without initial conditions.
    • Practice Problems - Section 1.3: 9, 11, 14, 17, 27
  • LO2: I can use a slope field to answer questions about a first order ODE.
    • Practice Problems - Section 1.1: 1, 3, 11, 13, 15, 17, 25, 27
  • LO3: I can find the general solution to a first order ODE using the method of integrating factors.
    • Practice Problems - Section 2.1: 1, 5, 11, 15, 21, 25
  • LO4: I can find the general solution to a separable first order ODE.
  • LO5: I can apply the best method to find a solution to first order ODEs/IVPs and describe behavior. (CORE)
  • LO6: I can build and analyze a first order IVP to model a given application.
  • LO7: I can determine if and when a unique solution exists for a first order ODE.
  • LO8: I can classify the equilibria of an autonomous first order ODE, and describe the long term behavior of solutions. This includes finding equilibria by hand or from a direction field, as well as an understanding of bifurcations. (CORE)

Second Order Differential Equations
  • LO10: I can determine if a set of functions form a fundamental set of solutions to a second order ODE.
  • LO11: I can find the general solution to a homogeneous second order constant coefficient ODE.
  • LO12: I can apply the appropriate technique to find the general solution to a nonhomogenous second order constant coefficient ODE.
  • LO13: I can determine when a unique solution exists for a second order linear ODE.
  • LO14: I can model (free or forced, damped or undamped) mechanical oscillators with a second order ODE and analyze the system.
  • LO15: I can use power series to find a series solution of a second order ODE near given point.
  • LO16: I can use the Laplace Transform method to find the solution of second order IVPs with discon- tinuous forcing function.
  • LO17: I can use the Laplace Transform method to find the solution of second order IVPs with impulse forcing function.
Systems of ODEs
  • LO18: Conversions of higher order differential equations. I can convert between a higher order ODE and a first order linear system.
  • LO19: I can solve IVPs for systems of constant coefficient homogeneous linear ODEs.
  • LO20: I can analyze a system of linear differential equations using a phase plane.
  • LO21: I can classify the equilibrium for linear autonomous ODE systems as well as determine the geometric behavior of the system.
  • LO22: I can analyze the stability of a system of nonlinear differential equations using a phase plane.
  • LO23: Given a system of differential equations, I can compute the Jacobian matrix.
  • LO25: I can find and classify equilibria for nonlinear autonomous ODE systems, as well as determine
    • the geometric behavior of the system.
  • LO26: I can analyze the dynamics of two interacting populations with a system of ODEs.
Habits of Mind
  • LO27: Preparation and Participation. I accumulated at least 80% of available participation points, through pre-class reading quizzes, participating in class and working in groups.
  • LO28: Practice. I fully completed at least 80% of homework assignments.
